# gocsv

### go build
#### Download
https://github.com/midorigreen/gocsv/releases/tag/v0.1

#### build
```
% go get github.com/midorigreen/gocsv
% go build
```

### go run
```
% ./gocsv -c 3:4 -f jacoco.csv -g [search word]
```

## Shell Script
```
# ver jacoco.csv
% sh jacoco.sh [file: search word row] [csv file]
```
